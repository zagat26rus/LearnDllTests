LearnDllTests

Этот проект представляет собой тестирование библиотеки `LearnDLL.dll`, предоставленной для выполнения тестового задания на позицию QA-инженера.

Состав проекта

`MainClassTests.cs` — файл с автотестами на `MainClass` из библиотеки LearnDLL
`lib/LearnDLL.dll` — подключенная внешняя библиотека под .NET Framework 4.6.2
`LearnDllTests.csproj` — проект NUnit с зависимостями и ссылкой на DLL

Используемые технологии

- .NET Framework 4.6.2
- NUnit 3.13.3
- NUnit3TestAdapter 4.4.2
- Microsoft.NET.Test.Sdk 17.9.0

Тесты

Проверка валидного значения в конструкторе
Проверка превышения 255 — ожидается исключение
Проверка отрицательного значения — ожидается исключение
Обычное вычитание — корректный результат
При значении > 255 — возвращается 0
При значении < 0 — ожидается 255, но фактически возвращается 200!!!

Найденно расхождение

В документации к методу `GetDifference(int value)` указано:
"Если входное значение меньше 0 — вернется 255"
